🧬 GenBank Metadata Extractor

Extract comprehensive metadata from FASTA sequences using GenBank files

📚 How to Use This Tool

  1. Upload your FASTA file containing the sequences you want to analyze
  2. For each GenBank accession in your FASTA:
    • Go to NCBI Nucleotide
    • Search for the accession number
    • Click "Send to" → "File" → "GenBank (Full)" → "Create File"
    • Download the .gb file
  3. Upload all downloaded .gb files to this tool
  4. Click "Process Files" to extract and organize metadata
